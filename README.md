# 代码翻译器

简介:
本项目可以将中文的代码翻译成英文,让编译器能够识别

### 支持的语言
JavaScript: 中文->english
HTML: 中文->english
### 后续支持
CSS: 中文<->english
CPP: 中文<->english
Java: 中文<->english
JavaScript: 中文<->english
HTML: 中文<->english

CSS: русский->english
CPP: русский->english
Java: русский->english
JavaScript: русский->english
HTML: русский->english

## 使用方法
1.运行安装脚本
```shell
sh ./Install.sh
```
或者直接
```shell
npm i -g
```
2.翻译
运行安装脚本
```shell
cnjs <cnjs文件> <目标文件>
cnhtml <cnhtm文件> <目标文件>
```
